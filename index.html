How to Get SSL/HTTPS for Localhost

Introduction

In web development, maintaining a protected connection between the client's browser and the server is crucial. To achieve this security, SSL/TLS (Secure Sockets Layer/Transport Layer Security) is used. SSL/TLS employs encryption to establish a secure channel between the client and the server, guaranteeing that all transmitted data remains private and unaltered. This article will guide you through the process of obtaining SSL/HTTPS for localhost.

What is SSL/HTTPS?

SSL/TLS is a method used by web browsers and servers to create a secure connection. It keeps information sent between them hidden and protected from changes. To do this, SSL/TLS uses two types of cryptography: public key cryptography, which creates a unique key for each user, and symmetric key cryptography, which uses a single key shared between the two parties.

There are several ways to get SSL/HTTPS for localhost. Here are a few methods:

Method 1: Self-Signed Certificate

To enable SSL/HTTPS on your localhost, you can create a self-signed certificate. This type of certificate is created and signed by your own computer trusted Certificate Authority (CA)。You can use tools like OpenSSL to create self-signed certificates.

STEPS:

✔️.Open a terminal and run the following command: openssl req -x509 -newkey rsa:2048 -nodes -keyout localhost.key -out localhost.crt -days 365
✔️.Fill in the required information, such as your name, organization, and email address. Run the following command to generate the certificate: openssl x509 -req -days 365 -in localhost.csr -signkey localhost.key -out localhost.crt

Method 2: Let's Encrypt

Let's Encrypt offers no-cost SSL/TLS certifications that are suitable for use in software development and testing environments.the steps are as follows

Install the Let's Encrypt client on your system: sudo apt-get install letsencrypt, Run the following command to obtain a certificate: letsencrypt certonly --webroot --webroot-path=/var/www/html -d localhost,Follow the prompts to complete the certificate request.

INSTALLING ELASTICSEARCH

Elasticsearch is a popular open-source search and analytics engine built on top of Apache Lucene,it is the core component of ELK that stores and indexes log data. To install Elasticsearch, follow these steps:

✔️Open a terminal and run the following command: sudo apt-get update

✔️Install Elasticsearch using the following command: sudo apt-get install elasticsearch

✔️Start Elasticsearch using the following command: sudo systemctl start elasticsearch

✔️Verify that Elasticsearch is running by visiting http://localhost:9200 in your web browser
